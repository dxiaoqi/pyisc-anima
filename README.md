# pyisc-anima
## canvas的物理动画库
![](https://img.shields.io/badge/powerby-%E5%B1%B1%E9%AC%BC-yellowgreen.svg) ![](https://img.shields.io/github/license/mashape/apistatus.svg)

## 特性
1. 支持canvas,采用离屏渲染
2. 支持多种动画交互
> 这是一份食用性还不错的小东西，你可以使用它来快速构建你的小游戏，以后可能还会支持VUE 小程序等，同时你也可以利用它学习很多有关动画方面的知识，
> 可以说它会是一款小型的物理引擎。
